# Code of Conduct

## Adoption

This project adopts the **Contributor Covenant 2.1** as its Code of Conduct.

The full text is the canonical reference and lives at:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

Translations are available at:

https://www.contributor-covenant.org/translations/

By participating in this project (issues, PRs, discussions), you agree to follow that document in full.

## Scope

Applies to all project spaces (GitHub repository, issues, pull requests, discussions) and to public spaces when an individual is officially representing the project.

## Reporting

If you observe behavior that violates the Code of Conduct, please report it via one of:

- **GitHub Security Advisories** (private, preferred for sensitive reports): https://github.com/omardimarzio/GlueZero/security/advisories/new
- **Email**: `conduct@<TBD>` (replace with project contact once domain is registered)

Reports will be reviewed in a timely manner. The maintainers will respect the reporter's privacy and will not retaliate against good-faith reporters.

## Enforcement

Maintainers are responsible for clarifying the standards of acceptable behavior and are expected to take appropriate corrective action in response to any behavior they deem inappropriate. They have the right to remove, edit, or reject comments, commits, code, issues, and other contributions that do not align with this Code of Conduct.

The Contributor Covenant 2.1 [Enforcement Guidelines](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines) define the consequence ladder used by maintainers (correction → warning → temporary ban → permanent ban).
