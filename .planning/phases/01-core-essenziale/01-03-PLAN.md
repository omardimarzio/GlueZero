**1. Creare `packages/core/src/types/deep-readonly.ts` (verbatim da RESEARCH.md):**

```ts
export type DeepReadonly<T> =
  T extends Date | RegExp | Error
    ? T
    : T extends Map<infer K, infer V>
      ? ReadonlyMap<DeepReadonly<K>, DeepReadonly<V>>
      : T extends Set<infer U>
        ? ReadonlySet<DeepReadonly<U>>
        : T extends Array<infer U>
          ? ReadonlyArray<DeepReadonly<U>>
          : T extends object
            ? { readonly [K in keyof T]: DeepReadonly<T[K]> }
            : T
```

Rationale: D-07 (branded immutable types). Date/RegExp/Error sono passthrough (no recursion). Map/Set diventano Readonly varianti.
