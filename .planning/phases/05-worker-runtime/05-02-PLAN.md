Pattern: `assertSerializable(value, rootPath = '')` chiama `walk(v, path, visited)` con `WeakSet` per cycle detection. `walk` ha branches:
1. Primitives + null/undefined: return (SCA OK)
2. function: throw `createBrokerError({ code: 'worker.serialization.failed.function', message: 'Field at "...path..." is a function, not SCA-cloneable...', category: 'worker', details: { fieldPath: path, fieldType: 'function' } })`
3. symbol: throw analog con code `worker.serialization.failed.symbol`
4. object: cycle check via `visited.has(v)` → return; poi `visited.add(v)`. Sub-branches:
   - DOM node (`v instanceof Node`): throw `worker.serialization.failed.dom-node`
   - SCA-native built-ins (`ArrayBuffer | Date | RegExp | ArrayBuffer.isView | Blob | File | ImageData | MessagePort`): return
   - Map: recurse on entries con path `[Map.key#i]` / `[Map.value#i]`
   - Set: recurse con path `[Set#i]`
   - Array: `for i = 0; i < v.length` recurse con path `${path}[${i}]`
   - Plain object: check `Object.getPrototypeOf(v)` — se `null` o `Object.prototype` recurse on keys con path `path === '' ? key : path + '.' + key`. ALTRIMENTI throw `worker.serialization.failed.custom-class` con `proto?.constructor?.name`.
5. Fallback unreachable: throw custom-class con `fieldType: t`.

Message del throw deve includere fieldPath e suggested fix per `function` (registrare via `registerTransform(name, fn)` e passare `transformId`).

**2.2 — GREEN phase: scrivi `packages/worker/src/transferable-extractor.ts`.**

Signature: `export function extractTransferables(payload: unknown, paths: readonly string[]): Transferable[]`

Algoritmo:
1. Early return `[]` se `paths.length === 0`
2. `const collected = new Set<Transferable>()` per dedup
3. Per ogni path: `const segments = parsePath(path); if (segments === null) continue; walk(payload, segments, 0, collected)`
4. Return `[...collected]`

`parsePath(path: string): Segment[] | null` con:
- Type `Segment = { type: 'key'; name: string } | { type: 'wildcard' }`
- Validazione: rifiuta `path === ''`, `'..'`, leading/trailing `.` → ritorna null
- Loop char-by-char: accumula `buf`, su `.` push `{ type: 'key', name: buf }`, su `[` aspetta `*]` literal e push `{ type: 'wildcard' }` (V1 supporta solo wildcard, no numeric index)
- Malformed bracket: ritorna null

`walk(current, segments, segIdx, collected)`:
- Se current null/undefined: return
- Se segIdx === segments.length: `if (isTransferable(current)) collected.add(current)`; return
- Wildcard segment: se Array.isArray(current), per ogni item `walk(item, segments, segIdx+1, collected)`
- Key segment: se typeof current === 'object', `walk((current as Record<string, unknown>)[seg.name], segments, segIdx+1, collected)`

`isTransferable(v)`:
- Check `v instanceof ArrayBuffer` || `MessagePort` (con typeof guard) || `ImageBitmap` (typeof guard) || `OffscreenCanvas` (typeof guard)
- NOT TypedArray direct (TypedArray non è transferable, ma `TypedArray.buffer` è — il path target deve essere `.buffer` per TypedArray cases)
