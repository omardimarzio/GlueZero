Weather widget — canonical mapping demo

Two plugins. Different naming. They never talk to each other directly. The Italian-named form publishes weather.requested with fields città data. The English-named widget subscribes to weather.loaded and expects location forecast-date temperature. GlueZero translates both directions through the canonical model. A mocked HTTP route (no real network) handles the request with optional cache + simulated failure.
